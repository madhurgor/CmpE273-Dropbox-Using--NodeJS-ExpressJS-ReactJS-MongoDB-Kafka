Open two terminals and follow the steps on the terminals to start the servers:

Kafka server
	1. cd kafka-back-end
	2. npm install
	3. npm start

Back-end server
	1. cd back-end
	2. npm install
	3. npm start

Front-end server
	1. cd front-end
	2. npm install
	3. npm start

-->upload folder needs to be created in Main Directory if it does not exist, without this folder application will not work..
